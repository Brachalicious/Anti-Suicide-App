# Mental Health Support & Suicide Prevention App

## Project Overview
A comprehensive mental health support application focused on suicide prevention, providing:
- Crisis intervention resources and hotlines
- Mood tracking and journaling
- Supportive community features
- Professional help connections
- Emergency contact system
- Wellness activities and coping strategies

## User Preferences
- Focus on accessibility and sensitivity
- Clean, calming design with gentle animations
- Mobile-first responsive design
- Privacy and security as top priorities

## Project Architecture
- **Frontend**: React with TypeScript, Wouter routing, TanStack Query
- **Backend**: Express.js with in-memory storage
- **Styling**: Tailwind CSS with shadcn components
- **Data**: In-memory storage for user sessions and mood tracking

## Key Features
1. **Crisis Support**: Immediate access to helplines and emergency contacts
2. **Mood Tracking**: Daily mood check-ins with visualization
3. **Journal**: Private journaling with optional sharing
4. **Resources**: Educational content and coping strategies
5. **Community**: Anonymous peer support (if requested)
6. **Professional Help**: Directory of mental health professionals

- All data kept private and secure
- Option for anonymous usage
